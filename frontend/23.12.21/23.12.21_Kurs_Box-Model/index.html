Box Model

Box Model beschreibt, wie sich Blockelemente auf einer Seite verhalten.

Der Platz errechnet sich aus:

  1. width / height (Content-Box)
  2. padding (Padding-Box)
  3. border (Border-Box)
  4. margin (Außenabstand zwischen Elementen, Margin-Box)

Box-sizing property

hilft uns, Box Model zu manipulieren

box-sizing:content-box (default Wert) => die Boxen werden aufaddiert

box sizing:border-box => Breite wird automatisch mit Border verrechnet

Margin und Padding

Margin

ist der Außenabstand

verschiedene Schreibweisen:

  • margin: 10px; => gilt für alle Seiten
  • margin: 10px 10px; => oben/unten und links/rechts
  • margin: 10px 10px 20px 20px; => das ist dann im Uhrzeigersinn von oben ausgehend
  • margin-top: 10px; => Oberseite
  • margin-right: 10px; => rechte Seite
  • margin-bottom: 20px; => Unterseite
  • margin-left: 20px; => linke Seite

Padding

ist der Innenabstand

Display property: inline, block, inline-block

inline: Elemente können Höhe und Breite annehmen

block: nehmen 100% der Breite ein

inline-block: kombiniert das Beste von beiden, man hat nicht immer die 100% & man kann trotzdem eine Höhe und Breite vergeben; sehr gut für responsive Designs
