3 Möglichkeiten, CSS in HTML einzubinden

1. Externe Datei ✅

Über >link rel="stylesheet" href="./style.css" /< in der Head wird die Verknüpfung zum css-Dokument angelegt

2. Intern

Innerhalb des Tags style in der Head kann via CSS gestylet werden. Aber wegen der besseren Übersichtlichkeit ist es in der Regel besser, CSS auszulagern in eine eigene Datei.

3. Inline Styling

Das Styling wird direkt in den Tag geschrieben, den ich stylen möchte, zB: >h3 style="background-color: red;"< stylt ausschließlich diese eine H3 rot.

Selektorspezifität

  • Weiteres dazu
  • bei Style Konflikten in CSS gewinnt der Befehl, der die niedrigeren Zahlen hat (0, 1, 3)
  • mit !important kann diese Hierarchie aufgehoben werden (aber nicht best practice)
